# 청약 인텔리전스

청약통장으로 아파트 청약 시 **당첨 가능성 분석 · 가점 진단 · 분양현장 알리미 · 사후관리**를 제공하는 정적 웹앱.
빌드 도구·의존성 없음 (순수 HTML/CSS/JS).

## 페이지
| 파일 | 내용 |
|---|---|
| `index.html` | 홈 — 청약 5단계 여정 다이어그램 + 분양현장 알리미(경쟁률·필터) |
| `page2.html` | 청약 준비 — 가점 계산·자금 리포트·셀프체크 + 기준표(예치금/가점/체크리스트11) |
| `manage.html` | 내 청약 관리 — 청약 기록·낙첨 리뷰·다음 일정 추천 |
| `board.html` | 게시판 — 공지 55건 + FAQ (탭·필터·검색) |

## 실행
> ⚠️ `file://` 더블클릭은 데이터 JS 로딩이 막힐 수 있으니 **로컬 서버**로 실행하세요.

```bash
# 폴더에서
python -m http.server 8777
# 브라우저: http://localhost:8777/index.html
```

## 데이터
- `bunyang.js` — 분양현장 라이브 데이터(청약홈 수집, `window.BUNYANG`)
- `board-data.js` — 공지/FAQ(`window.BOARD`)
- `kb.js` / `matching.js` — 전략·절차 지식베이스, 가점·매칭 로직
- 개인 입력값(가점·자금·체크·기록)은 **localStorage**에만 저장(서버 전송 없음)

`청약자료/` — 청약홈 분석 자료집(여정 정리·예치금표·가점표·체크리스트 원본).
