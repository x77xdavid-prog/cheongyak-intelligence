# cheongyak.mapsaihelp.com 배포 안내 (Render Static Site)

선정: **blueprint(도면) 모드**. 이 `site/` 폴더가 자체완결 배포물(index.html + bunyang/matching/kb.js).
운영중 210 부동산 콘솔은 **전혀 건드리지 않음** (별도 서브도메인).

## 1) Render Static Site 생성
1. https://dashboard.render.com → **New +** → **Static Site**
2. GitHub 연결 → `x77xdavid-prog/cheongyak-intelligence` 선택
3. 설정:
   - **Branch**: `main`
   - **Build Command**: (비움)
   - **Publish Directory**: `site`
4. **Create Static Site** → 자동 배포 후 `https://<이름>.onrender.com` 발급. 열어서 동작 확인.

## 2) 커스텀 도메인 연결
1. 해당 Static Site → **Settings → Custom Domains → Add** → `cheongyak.mapsaihelp.com`
2. Render가 보여주는 **CNAME 대상**(예: `<이름>.onrender.com`) 복사
3. DNS(mapsaihelp.com는 **Cloudflare**)에서 레코드 추가:
   - Type **CNAME**, Name `cheongyak`, Target `<이름>.onrender.com`
   - Cloudflare면 처음엔 **Proxy 끄기(DNS only, 회색 구름)** 권장 → SSL 발급 후 필요시 켜기
4. Render가 도메인 검증 + SSL 자동 발급(수 분~수십 분).

## 끝나면
`https://cheongyak.mapsaihelp.com` = blueprint 청약 인텔리전스(공개). 운영 부동산 콘솔과 완전 분리.

## 갱신 방법
이 repo `site/` 갱신 후 push → Render 자동 재배포. 데이터(bunyang.js) 갱신도 `site/bunyang.js` 교체 후 push.
