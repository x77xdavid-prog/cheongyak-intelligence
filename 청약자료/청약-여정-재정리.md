# 청약 여정 (가입자 관점 재정리)

> **이 여정 한 문장 요약:** 청약통장부터 만들고 → 내 점수·자격을 확인하고 → 들어갈 수 있는 단지를 찾아 → 정해진 시간에 신청하면 끝. (모든 최종 기준은 그 단지의 입주자모집공고문이 우선)

---

## 1. 통장 만들기 — 청약의 입장권인 청약통장을 연다. 없으면 시작도 못 한다.
- ✅ **할 일:** 9개 은행(농협·신한·우리·하나·기업·국민·아이엠뱅크·부산·경남) 중 한 곳에서 **주택청약종합저축** 가입. 매월 2만~50만원 자유 납입 시작.
- 📋 **필요한 것:** 본인 실명확인증표. (나이·자격 제한 없이 누구나 가능)
- ➡️ **다음 조건:** 통장 개설 완료 → 2단계. *(이미 있으면 바로 2단계)*
- 💡 가입기간이 곧 점수다. 청약할 집이 없어도 **빨리 만들수록 유리.**

## 2. 내 점수·자격 확인 — 내가 1순위인지, 가점이 몇 점인지 미리 안다.
- ✅ **할 일:** ① 청약홈 '청약가점 계산하기'로 내 점수 직접 계산(84점 = 무주택기간 32 + 부양가족 35 + 통장 가입기간 17). ② 1순위 가입기간 확인 — 규제지역(투기과열·청약과열) 2년 / 그 외 수도권 1년 / 수도권 외 6개월.
- 📋 **필요한 것:** 주민등록표등본(세대·거주지), 통장 순위기산일·납입 실적. (민영은 면적·지역별 예치금도)
- ➡️ **다음 조건:** 순위·점수 확정 → 3단계. *(점수는 본인이 직접 계산, 틀리면 당첨 취소될 수 있음)*
- 🔀 **갈림길:** 가점 높으면 → **가점제 정공법(85㎡ 이하 민영 등)** / 가점 낮으면 → **추첨제·특별공급**(3·4단계에서 노린다).

## 3. 이 집이 내 조건에 맞나 — 단지의 주택종류·규제 여부로 신청 가능한지 거른다.
- ✅ **할 일:** ① 국민주택(공공 85㎡ 이하, **무주택세대구성원만**)인지 민영주택인지 확인. ② 규제지역(투기과열·청약과열)인지 확인 → 규제지역이면 1순위 요건 강화(2년 / 국민주택 납입 24회) + 제한자 규정 발동.
- 📋 **필요한 것:** 등본상 거주지·세대주 여부, 과거 5년 내 당첨 이력, 보유 주택 수.
- ➡️ **다음 조건:** 자격 충족 → 4단계.
- 🔀 **갈림길:** **1순위 제한자**(세대주 아님 / 5년 내 당첨된 세대에 속한 자 / 2주택 이상 소유 세대)는 1순위라도 **2순위로 신청.** *(현재 규제지역: 서울 전역·경기 일부)*

## 4. 신청할 단지 찾기 — 청약 중인 단지를 고르고 일정을 챙긴다.
- ✅ **할 일:** ① 청약홈 '분양정보/경쟁률'에서 화면 선택 — 일반 아파트 **APT**, 무순위·잔여세대 **APT 잔여세대**, 오피스텔·생숙·임대는 별도 화면. ② 모집공고일·청약기간·당첨자발표일 확인 후 주택명 눌러 **입주자모집공고문** 확인.
- 📋 **필요한 것:** 관심 지역·주택구분 필터, **[알림 신청]** 등록.
- ➡️ **다음 조건:** 넣을 단지·트랙(가점제/추첨제/특공) 확정 → 5단계.
- 💡 **잔여세대(무순위)는 청약기간이 하루뿐인 경우가 많다** — 알림 신청은 사실상 필수. 무순위 잔여세대는 무주택세대구성원(성년자)이면 통장·순위·가점 없이도 신청 가능.

## 5. 신청하고 결과 확인 — 정해진 시간에 신청하고, 발표일에 결과를 본다.
- ✅ **할 일:** ① 청약기간 중 **09:00~17:30** 청약홈에서 인터넷 신청. ② 발표일에 '청약당첨조회'·'신청내역조회'로 결과 확인.
- 📋 **필요한 것:** 본인 명의 인증서 1개(공동·금융·네이버·KB국민·토스·신한·카카오). 단 **무순위·임의공급·불법행위재공급은 공동·금융인증서만.** + 청약통장.
- ➡️ **다음 조건:** 당첨 시 사업주체가 자격 서류 사후 확인(부적격이면 취소 가능). 입주 시기는 '입주(예정)정보' 참고.
- 💡 **당일 17:30 전이면** 취소 후 재신청 가능, 마감 후엔 정정·취소 불가. 정보취약계층은 은행 지점에서 1·2순위만 가능(특별공급 불가).

---

> **공통 철칙:** 신청 직전 반드시 **그 단지의 입주자모집공고문을 직접 확인**한다. 위 수치는 참고용이며 최종 기준은 개별 공고문이 우선. 막히면 청약문의 **1644-7445**(평일 09:00~17:30).

---

### 유형별 빠른 길
- **고가점(무주택 장기 가입) → 가점제 정공법:** 가점이 통하는 단지를 노린다. 투기과열지구 85㎡ 초과는 가점 80%·60㎡ 초과 85㎡ 이하는 70%로 고가점에 유리. 점수보다 **거주지 불일치·세대주 요건·5년 내 당첨 이력**(2·3단계)이 더 큰 탈락 위험이니 신청 전 제거.
- **저가점(사회초년생·신혼) → 추첨·특공:** ① 무주택이면 **특별공급**(청년·신혼부부·생애최초·신생아 등)을 길로. ② 일반공급은 **추첨 비중 큰 민영**(85㎡ 초과 비규제 = 추첨 100%, 85㎡ 이하 = 추첨 60% 이상, 무주택 추첨 75% 우선). ③ 떨어지면 **무순위 잔여세대**로 재도전.

## 검증 메모

**(a) 교정한 항목**
1. **2단계 — "통장 가입일·납입 실적" → "통장 순위기산일·납입 실적".** 자료집 가점계산기 입력 항목은 '순위기산일·생년월일'(B-3, line 240)이라 정확한 용어로 교정.
2. **3단계 — "세대 전원 무주택만" → "무주택세대구성원만".** 자료집 C-2(line 289)의 공식 용어로 통일.
3. **3단계 — "1순위 요건 강화(2년·24회)" → "(2년 / 국민주택 납입 24회)".** 24회는 국민주택 납입횟수 요건이고 민영은 예치금 기준(C-3, line 312)이므로, 24회가 모든 주택에 일률 적용되는 듯한 표현을 분리·명확화.
4. **3단계 갈림길 — "5년 내 당첨 세대 / 2주택 이상" → "5년 내 당첨된 세대에 속한 자 / 2주택 이상 소유 세대".** 자료집 C-4(line 320) 원문 표현에 맞춰 정밀화(의미 동일, 오해 방지).
5. **4단계 💡 — "무주택 세대구성원이면 …" → "무순위 잔여세대는 무주택세대구성원(성년자)이면 …".** 자료집 F-6(line 758)상 통장·순위 불필요는 '무순위'에 한정되며(임의공급은 사업주체가 별도 요건), 성년자 요건도 명시되어 있어 적용 범위를 한정.
6. **빠른 길 — "투기과열지구 85㎡ 초과는 가점 80%·이하는 70%" → "85㎡ 초과 80%·60㎡ 초과 85㎡ 이하 70%".** 자료집 F-3 비율표(line 609-610)상 70%는 60㎡ 초과 85㎡ 이하 구간이고 60㎡ 이하는 40%이므로, '이하=70%'의 부정확함을 구간 명시로 교정.
7. **빠른 길 — 특별공급에 "(청년·신혼부부·생애최초·신생아)"에 "등" 추가, "1순위 길로" → "길로".** 자료집 F-5는 특별공급이 7종+추가 유형이며 이 4종은 일부이고, '1순위'는 가점제 순위 개념과 혼동될 수 있어 완화.
8. **표현 다듬기 — "통장기간 17" → "통장 가입기간 17"(2단계), "17:30 전이면" → "당일 17:30 전이면"(5단계).** 자료집 용어(가입기간)·취소 조건(당일, line 892)과 일치.

**(b) 근거 없어 단정 못한 항목**
- **규제지역 "현재: 서울 전역·경기 일부"**: 자료집 C-5(line 334-335)는 서울 전역·경기 일부를 명시하나 지정·해제 시행일은 원문에 없음(line 348). 시점에 따라 변동 가능하므로 "현재"는 참고로만 둠.
- **예치금 구체 금액(예: 서울/부산 85㎡ 이하 300만원)**: 자료집 A-3은 청약예금·청약부금 기준 표(line 65-70)로 수록되어 있어 민영 청약 예치금 충족 판단에 쓰이나, 여정 본문에는 금액을 노출하지 않고 "면적·지역별 예치금"으로만 두어 공고문 우선 원칙을 유지(자료집도 최종 기준은 공고문이라 명시).
- **특별공급 세부 자격(소득·자산·청약통장 요건)**: 4단계·빠른 길에서 유형명만 언급. 자료집 F-5는 유형별로 요건이 상이하여 한 줄로 단정 불가, 의도적으로 일반화함.
