# 청약홈 로그인 게이트 페이지 구조 (인증 세션 추출)

> 로그인(간편인증) 세션으로 페이지 내부 fetch 추출. **개인정보 값은 제외하고 필드/컬럼 구조만** 기록. 출처 applyhome.co.kr.

## 청약제한사항확인 — `/co/cob/selectSubscrptLmttCnList.do`
청약 제한 10개 항목, 컬럼: `당첨된 주택의 구분 / 적용기간(당첨일로부터) / 민영주택 / 국민주택`
- 재당첨 제한
- 특별공급 제한
- 투기과열지구·청약과열지역 (1순위 청약제한)
- 가점제 당첨 제한
- 부적격 당첨자 제한
- 혼인특례 사용 제한
- 출산특례 사용 제한
→ 앱의 **결격 셀프체크**에 직접 매핑 가능.

## 주택소유확인 — `/ap/apd/selectHousePosesnList.do`
세대 주택소유 현황 조회 (이용시간·이용일 안내 + 조회 결과 테이블).

## 청약통장 가입내역 — `/ap/apd/selectSubscrptBnkbSbscrbDesc.do`
마이페이지 통장 내역. GET 직접호출은 빈 셸(데이터는 화면 내 AJAX 로드) → 별도 처리 필요.

## APT 당첨조회 — `/wa/waa/selectAptPrzwinDescList.do`
최근 5일 이내 당첨발표 주택. 컬럼: `성명 / 생년월일(주민번호 앞6) / 전화번호(뒷4) / 주택명 / 주택형 / 당첨구분 / 동·호수 / 당첨정보`
- 당첨자 발표일 16시부터 조회.

## 공공지원민간임대 당첨조회 — `/wa/wac/...` (= `/co/cob/selectPrvateRentPrzwinHouseList.do`)
컬럼: `주택명 / 건설 위치 / 당첨자 발표일`. 최근 5일 이내 당첨발표 주택 한정.

## 청약알리미 신청 — `/cu/cuc/selectSubscrptAllimiView.do`
입력 필드: `이름 / 휴대폰(3·4·4) / SMS 수신동의 / 인증번호 / 관심지역 유형 / 관심지역 지역`
신청내역 컬럼: `유형 / 지역 / 신청일자` · `주택명 / 모집공고일 / 청약기간`
관심유형: APT · 오피스텔/생숙/도시형/(공공지원)민간임대 · APT 잔여세대
→ 앱의 **청약 알리미/이메일 구독** 기능과 1:1 대응.
