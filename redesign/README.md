# redesign — 디자인 방향 제안 (2종)

기존 페이지(`index.html`, `page2.html`, `manage.html`, `board.html`)는 **그대로 두고**, 디자인 방향 제안 2종을 추가한 폴더입니다.
각 파일은 의존성 없는 **단일 HTML**(Pretendard + Google Fonts CDN, 외부 JS 없음). 브라우저로 바로 열어 비교할 수 있습니다.

| 파일 | 방향 | 요약 |
|---|---|---|
| `night.html` | 감정 야간 / 관제 터미널 | 다크 보드 + 거대 모노 숫자 + 청약 영수증·검증(VERIFIED) 인장. 빨강은 주 CTA 한 곳에만, 긍정=그린, 마감=앰버. |
| `blueprint.html` | 청사진 / 평면도 | 제도용지 + 테크니컬 라인 그리드 + 평면도/치수선 모티프. 제도 블루 + 도장 빨강 2색. 표 각 행에 유닛 플랜 썸네일. |

두 안 모두 공통 콘텐츠(오늘의 청약 원장 4지표 · 청약 5단계 · 분양현장 알리미 6행 · 내 가점 진단 · 마감 D-day 타임라인)를 담은 완성형 홈 시안이며, 데스크톱/모바일 반응형입니다.

> AI 보조 리디자인 시안입니다. 실제 데이터(`bunyang.js` 등) 연동은 포함하지 않은 정적 목업입니다.
